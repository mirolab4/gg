# 🌐 مولد جهات الاتصال الدولية

تطبيق ويب احترافي لتوليد أرقام هواتف بمقدمات 100 دولة مع دعم كامل لتصدير جهات الاتصال إلى Google Contacts.

## ✨ المميزات

- 🌍 **100 دولة** من مختلف القارات
- 📱 **مقدمات موزعين حقيقية** لكل دولة
- 🎨 **تصميم عصري** مع وضع داكن/فاتح
- 📤 **تصدير متعدد** (VCF, CSV, JSON)
- 🔍 **بحث وفلترة** سريع
- 📊 **إحصائيات مفصلة**
- 👥 **أسماء عشوائية** متنوعة
- 📱 **متجاوب** مع جميع الأجهزة
- ⌨️ **اختصارات لوحة المفاتيح**

## 🚀 النشر على GitHub Pages

1. ارفع الملفات إلى مستودع GitHub
2. اذهب إلى Settings → Pages
3. اختر Branch: `main` والمجلد: `/ (root)`
4. احفظ، وسيكون التطبيق متاحاً على:
   `https://username.github.io/repo-name/`

## 📖 الاستخدام

1. **اختر الدولة** من القائمة
2. **اختر الموزع** (أو جميع المقدمات)
3. **اضبط الإعدادات** (العدد، الأسماء، المجموعة)
4. **اضغط "توليد الأرقام"**
5. **صدّر** بالصيغة المناسبة

## 📤 رفع الملفات على Google Contacts

### ملف VCF (الأفضل):
1. اذهب إلى [contacts.google.com](https://contacts.google.com)
2. اضغط Import
3. اختر ملف `.vcf`
4. اضغط Import

### ملف CSV:
1. اذهب إلى [contacts.google.com](https://contacts.google.com)
2. اضغط Import
3. اختر ملف `.csv`
4. اضغط Import

## ⌨️ اختصارات لوحة المفاتيح

- `→` الخطوة التالية
- `←` الخطوة السابقة
- `Esc` الرجوع
